A responsive restaurant website template built with HTML, CSS, Bootstrap 5, and jQuery. Includes multi-page layout with menu, booking form, team, testimonials, and contact sections.

## Demo

![Restoran Preview](img/bootstrap-restaurant-template.jpg)

## Why This Project?

Most restaurant businesses lack a clean, professional web presence. This project provides a fully responsive, multi-page restaurant website that can be customized and deployed quickly — covering everything from food menus to table reservations.

## Features

| Feature | Description |
|---|---|
| Multi-page layout | Home, About, Menu, Service, Booking, Team, Testimonial, Contact |
| Responsive navbar | Sticky on scroll, dropdown on hover (desktop), collapse (mobile) |
| Food menu tabs | Breakfast, Lunch, Dinner tabs with item cards |
| Table booking form | Name, email, date/time picker, guest count, special request |
| Video modal | YouTube embed with autoplay on open |
| Testimonials carousel | Owl Carousel with autoplay and responsive breakpoints |
| Team section | Chef cards with social links |
| Animated counters | Years of experience, master chefs count |
| WOW.js animations | Scroll-triggered fade/zoom/slide animations |
| Back to top button | Smooth scroll to top |

## Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 / SCSS | Styling and layout |
| Bootstrap 5.0 | Responsive grid and components |
| jQuery 3.4.1 | DOM manipulation and event handling |
| Owl Carousel | Testimonials slider |
| WOW.js + Animate.css | Scroll animations |
| Tempus Dominus | Date/time picker |
| CounterUp.js | Animated number counters |
| Waypoints.js | Scroll position detection |
| Font Awesome 5 | Icons |
| Bootstrap Icons 1.4 | Additional icons |
| Google Fonts | Heebo, Nunito, Pacifico |

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mantrariziya/Restoran.git
   cd Restoran
   ```

2. Open in browser:
   ```bash
   # Simply open index.html in any browser
   open index.html
   # or double-click index.html in file explorer
   ```

No build step or server required — pure static HTML/CSS/JS.

## Security

- No backend or database — static site only
- No API keys or credentials in source code
- All external resources loaded via CDN (jQuery, Bootstrap, Google Fonts, Font Awesome)
